Questions¶

  1. Write a function that accepts an integer number of alligators as an input, and returns a string describing that number of alligators.

  2. Write a function that accepts an arbitrary number of integers, adds them together, and reports on that number of alligators (using the function you wrote in the first part).

  3. Write a function that converts a list of traffic counts into a list of traveler counts, including an optional automobile occupancy argument that defaults to 1.1 if not otherwise given.
